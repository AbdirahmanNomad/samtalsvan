# Contributing to Samtalsvän

Tack för ditt intresse att bidra! / Thanks for your interest in contributing!

---

## English

### How to Contribute

1. **Fork the repository** and create your branch from `main`
2. **Make your changes** with clear, descriptive commits
3. **Test your changes** locally before submitting
4. **Submit a pull request** with a description of your changes

### Development Setup

```bash
# Clone your fork
git clone https://github.com/YOUR_USERNAME/samtalsvan.git
cd samtalsvan

# Install dependencies
npm install

# Start development server
npm start
```

### Code Style

- Use clear, descriptive variable names
- Keep functions small and focused
- Add comments for complex logic
- Follow existing code patterns

### Accessibility Guidelines

This app is designed for elderly users. When contributing:

- **Large touch targets**: Minimum 48px, preferably 72px+
- **High contrast**: Ensure text is readable
- **Simple language**: Bilingual (Swedish/English) labels
- **Clear feedback**: Visual confirmation for actions

### Reporting Issues

When reporting bugs, please include:

1. Browser and version
2. Steps to reproduce
3. Expected behavior
4. Actual behavior
5. Screenshots if applicable

Open issues at: https://github.com/AbdirahmanNomad/samtalsvan/issues

### Feature Requests

Open an issue with:
- Clear description of the feature
- Use case / why it's needed
- Possible implementation approach

## Svenska

### Hur man bidrar

1. **Forka repositoriet** och skapa en gren från `main`
2. **Gör dina ändringar** med tydliga, beskrivande commits
3. **Testa dina ändringar** lokalt innan du skickar in
4. **Skicka en pull request** med beskrivning av dina ändringar

### Utvecklingsmiljö

```bash
# Klona din fork
git clone https://github.com/YOUR_USERNAME/samtalsvan.git
cd samtalsvan

# Installera beroenden
npm install

# Starta utvecklingsserver
npm start
```

### Kodstil

- Använd tydliga, beskrivande variabelnamn
- Håll funktioner små och fokuserade
- Lägg till kommentarer för komplex logik
- Följ befintliga kodmönster

### Riktlinjer för tillgänglighet

Denna app är designad för äldre användare. När du bidrar:

- **Stora pekmål**: Minimum 48px, helst 72px+
- **Hög kontrast**: Se till att text är läsbar
- **Enkelt språk**: Tvåspråkiga (svenska/engelska) etiketter
- **Tydlig feedback**: Visuell bekräftelse för åtgärder

### Rapportera problem

När du rapporterar buggar, vänligen inkludera:

1. Webbläsare och version
2. Steg för att reproducera
3. Förväntat beteende
4. Faktiskt beteende
5. Skärmdumpar om tillämpligt

Öppna ärenden på: https://github.com/AbdirahmanNomad/samtalsvan/issues

### Funktionsförslag

Öppna ett ärende med:
- Tydlig beskrivning av funktionen
- Användningsfall / varför den behövs
- Möjlig implementeringsmetod
